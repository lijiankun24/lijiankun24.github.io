写在2017年的初夏

Posted on 2017-06-26 | Views

如果真想看，请点击 Read more (阅读全文)

就在刚刚，我发现我收获了 GitHub 上面的第一颗 star，但是喜悦的心情只有一瞬，很快就又低沉下去了。还记得博客刚搭建起来的时候，发现自己的博客能多一个访问量都会高兴；也记得第一次向 掘金 投稿成功之后，收获第一个收藏，看着自己博客的访问量蹭蹭往上涨的时候(虽然现在访问量还是很小)，那两天的心情都是愉快的，天空都格外地蓝。按道理说在 GitHub 收获第一个 star，应该比前两次要更高兴，可是就是高兴不起来……

在18岁那年的夏天，我最后一次哭。当时正吃着饭，嘴就撇开了，眼泪也哗哗地流下来了，我记得还哭出了声，那可是在公众场合……我能想象得到当时我狼狈的样子。从那以后，我就暗暗对自己说：这辈子我都不会因为任何人任何事再流眼泪了。是的，至少到现在为止，我没有再流过眼泪，以后我也不想哭。可是有一种比自己哭更难受的情况…

一直见不得女人掉眼泪，就算是陌生的女人，在我面前哭泣，我也会替她感到委屈，更别说是自己熟知的女人了……当知道一个女人因为自己哭了，还哭过两次，我真切地体会到了什么叫做心如刀绞，我宁愿哭的那个人是我。

有人说，女人心软一肚子手，男人心软一肚子酒；也有人说，女人心软裤腰带松，男人心软一辈子穷，也许我会穷一辈子吧。有时候真地挺恨懦弱柔软的自己……
